# project-5

Hi, folks

Here, I am bringing a new project using Streamlit in Python. This project is simple and mainly aims to read and summarize the Copom Minutes. The project uses an LLM to summarize the Copom Minutes; moreover, it is worth noting that the project was proposed in Portuguese (my first language), so to read FED Minutes or other Central Banks, it is necessary to adapt the language or read in English. 

This project requires an API key, and obviously I hid my API key, so you have to put your own API key.
